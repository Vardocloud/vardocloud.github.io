## Örnek 1: SOL Analizi (10 Şubat 2026)

### Orijinal Mesaj
```
Solana direnç noktası 87,5$ seviyesi kırılmasıyla 94, 98$ 101$ seviyeleri
hızlı görünecek noktalar ardından 103.56$ seviyesi bir diğer kritik nokta
kırılmasıyla 115$, 120$ ve 128$ seviyeleri gelecektir

Orta vade hedef 240$ seviyesi bu piyasa için şimdilik

Alt destekler 80,74$ 78,56$ 73,15$ seviyeleridir

Btc hareketine dikkat edilmeli
Yatırım tavsiyesi değildir
```

### Yapısal Çözümleme
| Bileşen | Değer | Teknik |
|---------|-------|--------|
| İlk Direnç | 87,5$ | Breakout seviyesi |
| Kırılım Hedefleri | 94$ → 98$ → 101$ | Kademeli, hızlı hareket |
| Kritik Direnç | 103,56$ | "bir diğer kritik nokta" |
| 2. Kırılım Hedefleri | 115$ → 120$ → 128$ | Daha yavaş, güçlü direnç |
| Orta Vade | 240$ | Büyük hedef (2x+) |
| Destekler | 80,74$ → 78,56$ → 73,15$ | 3 kademeli |
| BTC Notu | ✓ | "Btc hareketine dikkat edilmeli" |

## Örnek 2: SUI Analizi (10 Şubat 2026)

### Orijinal Mesaj
```
Sui
direnç noktası 0,9821$
Kırılmasıyla beraber 1,15$, 1,28$ 1,48$ seviyelerini kısa süre içinde test edecektir
Zorlu direnç noktası 1,57$ seviyesi aşılmasıyla
2,10 2,32$ seviyelerine hareket izletir ardından 2,42$ seviyesi hacimli aşımıyla
2,95$ 3,20$ seviyesi beklenir orta vaadede hedef
4,50$ seviyesi
Alt destekler 0,901$ 0,8910$ 0,8388$ ve 0,76$ seviyesidir
Yatırım tavsiyesi değildir
```

### Yapısal Çözümleme
| Bileşen | Değer | Teknik |
|---------|-------|--------|
| İlk Direnç | 0,9821$ | "direnç noktası" |
| Kırılım (kısa) | 1,15$ → 1,28$ → 1,48$ | "kısa süre içinde" |
| Zorlu Direnç | 1,57$ | "zorlu direnç noktası" |
| Kırılım (orta) | 2,10$ → 2,32$ | "hareket izletir" |
| Hacimli Kırılım | 2,42$ → 2,95$ → 3,20$ | "hacimli aşımıyla" |
| Orta Vade | 4,50$ | Nihai hedef |
| Destekler | 0,901$ → 0,891$ → 0,839$ → 0,76$ | 4 kademeli |

**Kilit Teknik:** 3 farklı kırılım tipi kullanılmış:
1. Normal kırılım (0,98$ → 1,48$): "kırılmasıyla beraber"
2. Zorlu kırılım (1,57$ → 2,32$): "zorlu direnç aşılmasıyla"
3. Hacimli kırılım (2,42$ → 3,20$): "hacimli aşımıyla"

## Örnek 5: LDX Presale Başarı Hikayesi (26 Haziran 2026)

### Orijinal Mesaj
```
Evet 0.0012$ da risk alarak ön satışına girdiğimiz Ldx an itibariyle 3,13$
herkese hayırlı olsun
Bu tarz ön satışlar gelsin diyenler i görelim
👍🏼
yatırım tavsiyesi değildir
```

### Yapısal Çözümleme
| Bileşen | Değer |
|---------|-------|
| Giriş Fiyatı | 0,0012$ |
| Çıkış Fiyatı | 3,13$ |
| Getiri | ~260,833% |
| Strateji | Presale → çıkış |

**Kullanılan Teknik:**
1. **Sosyal Kanıt:** Başarılı çıkışı göstererek güven inşa etme
2. **Kıtlık:** "diyenler i görelim" — FOMO yaratma
3. **Risk Kabulü:** "risk alarak" — samimiyet
