## Orijinal Workflow (4 Adım)

Caption'daki tarif:

> "NotebookLM + Claude = gerçekten güvenebileceğin araştırma"
> "İkisi tek başına eksik: Claude akıl yürütür ama senin kaynaklarına dayanmaz;
>  NotebookLM kaynağa dayalıdır ama senin yerine yazmaz. Birleştir, döngüye sok:"

1. **① NotebookLM'i besle** → kaynak göstererek sentezlet (yalnızca senin kaynakların)
2. **② Sentezi Claude'a getir** → kıyas, çelişki, yapı, çerçeve
3. **③ Claude'a sor "ne doğrulanmalı?"** → iddiaları NotebookLM'in atıflarına geri götür (döngüyü kapat)
4. **④ Final sürümü Claude yazsın** → makale / thread / sunum / rapor

## Önemli Alıntılar

> "NotebookLM = getirme motoru (kaynağına sadık) · Claude = akıl yürütme + yazma motoru."

> "⚠️ Dürüst not: 'süper zeka / sıfır halüsinasyon' diye abartmaya gerek yok — kazanç şu:
>  NotebookLM kaynaklarının dışına çıkmaz + Claude düşünür/yazar; doğrulama adımı da kalanı yakalar."
