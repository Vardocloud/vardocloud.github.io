# Fırsatlar — Ücretsiz Tool/Provider Takip Sistemi

Gmail Pipeline ve Deep Dive sırasında keşfedilen ücretsiz ($0) araç, servis ve fiyatlandırma değişiklikleri.

## Kayıt Formatı

Her fırsat için `~/wiki/firsatlar/YYYY-MM-<slug>.md`:

```markdown
# [Araç/Provider Adı]
- Keşif: YYYY-MM-DD
- Kaynak: [hangi mail/link]
- Fiyat: $0 (ücretsiz)
- Kategori: [provider / tool / api / credit]
- Özellik: [1-2 cümle ne işe yarar?]
- Limit: [rate-limit, kota varsa]
- Link: [url]
- Kullanılabilirlik: [hemen / araştırma gerek / deneysel]
```

## Kategoriler

| Kategori | Örnek | Ne zaman kaydet? |
|----------|-------|------------------|
| **Provider** | Yeni ücretsiz LLM API'si | API endpoint + model listesi varsa |
| **Tool** | Ücretsiz kod asistanı, monitoring | Self-host edilebiliyorsa |
| **API** | Firecrawl free tier genişlediyse | Kota artışı/yeni özellik |
| **Credit** | OpenAI $5 credit, Cloudflare $10 | Kredi/deneme bakiyesi |

## Kurallar

- SADECE $0 (ücretsiz) olanları kaydet
- Paralı servisleri ATLA — "starting at $5/mo" bile olsa kaydetme
- "Free trial" olanları not et ama süresiyle birlikte
- Fiyat değişikliği varsa eski kaydı güncelle, yenisini oluşturma
- NotebookLM Tech Tools'a da ekle (etiket: "firsat")
