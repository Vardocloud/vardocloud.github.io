# Free AI Video Tools (Discovered July 2026)

Discovered from Instagram reel by @ceyhuntokgezer (https://www.instagram.com/reel/Da5_VjzoLNK/) and subsequent research.

## Vibes.ai (Meta)
- **URL:** vibes.ai
- **Status:** Currently FREE (may become paid)
- **Capabilities:** Text-to-video, voice overlay, music overlay, video extension
- **Interface:** Prompt input → "Add voice" → "Add music" → "Extend" workflow
- **⚠️ Caveats from comments:**
  - Heavy watermark on output
  - Expected to become paid service
  - "Metanın hiçbir şeyini kullanmayın" warnings from some users
- **Use case:** Quick prototyping, NOT production-ready for faceless channels

## nim.video
- **URL:** nim.video
- **Status:** Free tier available
- **Capabilities:** Video production acceleration, works alongside Claude/LLMs
- **Use case:** Faster video ideation and assembly

## nanobanana.io Seedance 2.0
- **URL:** https://nanobanana.io
- **Status:** Pro account required (Google sign-in)
- **Capabilities:** Image-to-video (2-10s), multi-image support, no audio
- **Models:** Seedance 2.0 (standard), Seedance Pro (best quality)
- **Use case:** Animating still illustrations for storytelling/social clips
- **Account:** Pro account at kenshin4155@gmail.com
## Google Gemini (Native Browser — PRIMARY Path)

- **URL:** https://gemini.google.com
- **Status:** Needs Google login (Pro/Advanced for full features)
- **Capabilities:**
  - **Image generation** via Nano Banana ("Görsel oluştur" feature in Gemini)
  - **Image-to-video** generation (3-8s clips from reference images)
  - **Image editing** with natural language prompts
- **Model names:** Nano Banana (standard), Nano Banana 2 (sharper), Nano Banana Pro (4K)
- **Account:** kenshin4155@gmail.com (Gemini Pro/Advanced)
- **Access method:** Browser automation — sign in, click "Yükleme ve araçlar" → "Görsel oluştur"
- **Use case:** PRIMARY path for image gen + short video clips. Best quality.
- **⚠️ Limitation:** Requires active login session; no programmatic API

## Pollinations Seedance / Veo (API)
- **URL:** https://gen.pollinations.ai (paid API with key)
- **Status:** Paid (API key required from Pollinations)
- **Capabilities:** Seedance 2.0 (2-10s, multi-image), Veo (4/6/8s, single-image, has audio)
- **Use case:** Programmatic integration in Python pipelines
