### 2. Keyword Kategorizasyon Çerçevesi

Her araştırmada keyword'leri şu kategorilere ayır:

| Kategori | Açıklama | CPC Beklentisi |
|----------|----------|----------------|
| **🔥 Yüksek Niyetli (High Intent)** | Hemen danışan arayanlar. "randevu", "arama", "ücret" içerir | Orta-Yüksek |
| **🧩 Uzun Kuyruk (Long Tail)** | 4+ kelime, spesifik. En yüksek dönüşüm, en düşük rekabet | Düşük |
| **📍 Lokal** | Şehir/semt ismi içerir. Coğrafi hedeflemeyle birleşince en düşük CPC | Düşük-Orta |
| **🌐 Online** | "online", "görüntülü", "zoom" içerir. Tüm Türkiye hedefi | Düşük |
| **🧪 Niş/Yöntem** | Spesifik terapi yöntemi (BDT, EMDR, şema). Düşük rekabet | Çok Düşük |
| **❓ Soru Formatı** | "nasıl", "ne yapmalıyım" ile başlar. Çözüm arayışı | Düşük |

### 3. Edel'in Özel Tercihleri (KRİTİK)

- **Standart keyword listeleri İSTEMEZ.** "anksiyete terapisi", "kaygı bozukluğu psikolog" gibi herkesin kullandığı kelimeleri zaten biliyor.
- **Yeni, alternatif, düşük rekabetli açılardan yaklaş.** Mevcut listeyi görünce "bunun dışında yeniler" talebi gelir.
- **Yaratıcı kategoriler dene:** semptom bazlı (teşhis koymadan arayanlar), halk dili (anksiyete demeyenler), fiziksel belirti bazlı, yaşam durumu kaynaklı.
- **Referanslı veri ver** — CPC benchmarkları, kaynak ismi, tarih. Uydurma rakam sunma.

### 5. Presence vs Interest Tuzağı (Sık Sorulan)

**Kullanıcı sorar:** "Lokasyon bazlı keywordler boş tıklama getirir mi?"

**Cevap:** Hayır, eğer kampanya ayarlarında **Presence** modu seçiliyse. Presence modunda:
- İzmir'de oturan + "bornova psikolog" yazarsa → reklam çıkar ✅
- İstanbul'da oturan + "bornova psikolog" yazarsa → reklam çıkmaz ❌
- İzmir'de oturan + sadece "kaygı terapisi" yazarsa → yine çıkar (lokasyon hedeflemesi sayesinde) ✅

Varsayılan "Presence or Interest" modu İstanbul'daki kişiye de reklam gösterir → BOŞ TIKLAMA.

**Lokasyon ismi keyword'de geçince Quality Score artar → CPC düşer.** Kullanılmalıdır.

### 7. Sağlık Reklam Politikaları (Google)

- **Kesin tıbbi iddialar YASAK:** "Depresyonunuzu kesin çözeriz" gibi ifadeler hesabı askıya aldırır
- **Remarketing sınırlıdır** — hassas kitle hedeflemesi kısıtlı
- **Empati ve güven odaklı dil** kullanılmalı
- **Reklam metninde:** gizlilik ilkesi, uzman bilgisi, ilk seans ücretsiz gibi unsurlar dönüşümü artırır

### 8. Halk Dilinde Anksiyete/Kaygı Alternatifleri (Türkiye'ye Özel)

İnsanlar "anksiyete" veya "kaygı bozukluğu" demeyebilir. Bunun yerine şunları kullanır:

- iç sıkıntısı
- sürekli endişe / endişelenme
- kuruntu / evham / vesvese
- sinir / sinirsel gerginlik
- aşırı düşünme (overthinking)
- iç daralması
- nedensiz korku / kötü bir şey olacak hissi

> **Bu kelimelere neredeyse hiçbir psikolog reklam vermez — çok düşük CPC, çok yüksek duygusal niyet.**

## Google Ads Programatik Erişim (API / MCP)

Google Ads hesabına programatik erişim için iki seçenek:

### Seçenek 1: Google Ads MCP Server (Resmi — Önerilen)
- **Kaynak:** `googleads/google-ads-mcp` (GitHub, 629⭐)
- **Ücret:** Tamamen ücretsiz, açık kaynak (Apache 2.0)
- **Yapabilir:** ✅ OKUMA (raporlama, analiz, performans sorgulama)
- **Tool'lar:** `search` (GAQL), `list_accessible_customers`, `get_resource_metadata`
- **Yapamaz:** ❌ YAZMA (campaign oluşturma/düzenleme/silme yok)
- **Auth:** Google Ads API developer token + OAuth Client ID/Secret
- **Kurulum:** pipx ile tek satır veya Docker/Cloud Run
- **Ne zaman kullanılır:** Sadece izleme, raporlama, analiz ihtiyacı varsa

### Seçenek 2: Composio Connect
- **Ücret:** Free tier 20K tool call/ay, $29/ay (200K call)
- **Yapabilir:** ✅ OKU + YAZ (campaign CRUD, ad group, customer list)
- **Tool'lar:** 9 tane — campaign mutate, ad group, customer list, GAQL
- **Auth:** Composio üzerinden OAuth (token yönetimi dahil)
- **Ne zaman kullanılır:** Reklam yönetimi (campaign açma/kapama, bütçe değiştirme) gerekiyorsa

### Önemli Farklar
| Özellik | Google Ads MCP Server | Composio |
|---------|----------------------|----------|
| Ücret | 🆓 Ücretsiz | $0-29/ay |
| Yazma | ❌ Read-only | ✅ Read+Write |
| Setup | Developer token + OAuth | Composio OAuth |
| Kontrol | Tam (kendi sunucun) | Üçüncü taraf |

### Karar Ağacı
- Sadece "performans nasıl, harcama ne kadar" → **Google Ads MCP Server** (ücretsiz, yeterli)
- Campaign yönetimi, bütçe değiştirme, teklif ayarlama → **Composio** (veya direkt Google Ads API)
- İkisi de olur → **Google Ads MCP Server** başla, ihtiyaç artınca Composio'ya geç

### Gereken Bilgiler (MCP Server Kurulumu İçin)
1. Google Cloud Project ID
2. Google Ads Developer Token (22 karakter, Google Ads API merkezinden)
3. OAuth Client ID + Client Secret (Google Cloud Console)
4. Google Ads Müşteri ID

Detaylı karşılaştırma ve kurulum notları için: `references/google-ads-programmatic-access.md`
