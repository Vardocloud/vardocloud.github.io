# Hakan Türkçapar Blog Yazı Üslup Rehberi

Bu dosya, Prof. Dr. Hakan Türkçapar'ın blog yazılarından öğrenilen üslup kurallarını içerir. 
LinkedIn postları bu üsluba uygun yazılmalıdır.

## Kimdir?

Prof. Dr. Hakan Türkçapar, Türk psikiyatr ve yazar. Psikoloji, psikiyatri ve toplumsal konularda 
blog yazıları yazmaktadır. Yazıları bilimsel ama erişilebilir, sıcak ve düşündürücüdür.

## Temel Üslup Kuralları

### 1. Doğrudan Giriş Yap
- "Bugün size X'ten bahsedeceğim" gibi giriş cümleleri YOK
- Konuya doğrudan gir, okuyucuyu yakala
- İlk cümle çarpıcı olmalı

**Örnek:**
```
❌ "Bugün sizlere sınav anksiyetesi hakkında bilgi vereceğim."
✅ "Sınav salonuna girdiğinizde kalbiniz hızla çarpmaya başlıyor, 
    elleriniz terliyor ve aklınızda tek bir soru dönüyor: 'Ya yapamazsam?'"
```

### 2. Kısa Paragraflar
- Her paragraf 2-4 cümle olsun
- Uzun bloklar okuyucuyu yorar
- Boşluklar nefes aldırır

### 3. Bilimsel Ama Erişilebilir
- Bilimsel terimleri açıkla
- "Psikologlar buna X diyor" gibi ifadeler kullan
- Teknik jargonu basitleştir
- Okuyucuyla diyalog kur

### 4. Retorik Sorular
- "Peki gerçekten öyle mi?"
- "Ne yapabiliriz?"
- "Bu durumda ne hissediyoruz?"
- Okuyucuyu düşünmeye davet et

### 5. Somut Kapanış
- Umut verici veya uygulanabilir öneri
- "Şimdi şu 3 adımı deneyin" gibi
- Boş vaatler değil, gerçekçi öneriler

### 6. Sıfır Emoji
- Emoji kullanma
- Güçlü kelimeler yeterli
- Profesyonel ama sıcak ton

### 7. Kaynak Göstergesi (Opsiyonel)
- Metin içinde: "Araştırmalar gösteriyor ki...", "Literatüre göre..."
- Link zorunlu DEĞİL
- Link verilecekse: yazının EN ALTINA, ayrı satır, sade URL
- Metin içinde link gösterme, kaynağı metne yedirme

## Cümle Yapıları

### Aktif Kullanım
```
❌ "Bu durumun birçok nedeni olduğu söylenebilir."
✅ "Bu durumun birkaç temel nedeni var."
```

### Somut İfadeler
```
❌ "Bir şekilde bu sorunla başa çıkabiliriz."
✅ "Bu 3 teknikle sorunla başa çıkabiliriz."
```

### Kişisel Dokunuş (Edilgen/Nesnel)
```
❌ "Klinik pratiğimde sıkça gördüğüm şu ki..."
❌ "Danışanlarımla yaptığım gözlem..."
✅ "Klinik gözlemler şunu gösteriyor ki..."
✅ "Literatürde bu konuda şu bulgular öne çıkıyor..."
```

### Etik Kurallar
- Birinci tekil şahıs ("ben", "gördüğüm", "danışanlarım") KULLANMA — edilgen/üçüncü şahıs yaz
- Bilim insanına isim/soyadı ile atıf yapılabilir ("Türkçapar'ın çalışmasında...", "X'e göre..."); yaşanmış gibi anlatma
- Bilimsel doğruluk: kanıta dayalı, abartısız
- Mahremiyet: gerçek vaka detayı verme, genelleştirilmiş gözlem kullan
- Zarar vermeme: umut ver ama gerçekçi ol, mucize vaat etme

```
[Sonuç - umut veya somut öneri, nesnel]

```

## Telif Önleme Kuralları

1. ASLA doğrudan alıntı YOK
2. Kaynağı oku → kapat → kendi kelimelerinle yaz
3. Çoklu kaynak birleştir
4. Kişisel klinik yorum ekle
