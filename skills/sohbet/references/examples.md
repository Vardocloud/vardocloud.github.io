## Curator Rules

1. **Read** all examples
2. **Group** similar examples by scenario/tactic
3. **Select** top 3-5 GOOD and 3-5 BAD per group (most representative)
4. **Extract** patterns using Structured Pattern Extraction
5. **Write** patterns to `patterns.md`
6. **Move** remaining examples to `archive/`
7. **NEVER** modify SKILL.md

### Curator Prompt

```
Review references/examples.md. If <5 examples, skip.
Group similar examples. Keep top 3-5 GOOD and 3-5 BAD.
Move rest to references/archive/.
Run structured pattern extraction (taktik success rates + Edel reaction patterns).
Write patterns to references/patterns.md.
Update SKILL.md Patterns section only.
```

### Archive Rules

- Examples older than 30 days → `archive/`
- Archive is pushed to "Conversation Archive" notebook on NotebookLM monthly
- After push, archive is cleared
