# NotebookLM Organizasyon Kuralları

## Aktif Notebook'lar ve Amaçları

| Notebook | Amaç | Karıştırma |
|---|---|---|
| 🔗 **GitHub Repo Arşivi** | GitHub repoları (inceleme, referans) | ❌ Başka şey atma |
| 🧠 **Vanitas Hafıza Arşivi** | Vanitas'a özel referanslar | ❌ GitHub reposu atma |
| 📚 **Bardo Araştırma Arşivi** | Bardo ile ilgili araştırmalar | |
| 🛠️ **Tech Tools Updates** | Araç güncellemeleri | |

## Kural (31 Mayıs 2026)

Edel düzeltmesi: "GitHub repoları diye açsan daha iyi olmaz mıydı tüm arşiv buraya mı kaydediliyor?"

→ GitHub repoları için **ayrı notebook** aç. Her şeyi "Vanitas Hafıza Arşivi"ne atma.
→ Yeni bir kaynak türü geldiğinde sorgula: "Bunun için ayrı notebook mu açsam?"
